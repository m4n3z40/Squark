# React Typescript Starter

A Electron App boilerplate that hooks up Typescript, React, Redux & Photon.

### Development

To build the source:
```sh
npm run build
```

To run the app (requires built source):
```sh
npm run app
```

To start watchers:
```sh
npm run watch
```

To start watchers and run app:
```sh
npm start
```
